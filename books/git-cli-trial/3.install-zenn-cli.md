## Zenn CLIのインストール

https://zenn.dev/zenn/articles/install-zenn-cli


> Zenn CLIを使うにはNode.js 14以上が必要です。


> Zennのコンテンツを管理したいディレクトリで、以下のコマンドを実行します。

```
% node -v # バージョンを確認
% npm init --yes # 初期化
% npm install zenn-cli # CLIのインストール
% npx zenn init # セットアップ
```
